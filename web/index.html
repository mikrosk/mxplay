About

mxPlay is an audiomedia player for Atari computers. Features modern GUI, support for long names, drag&drop, playlist, plugin based API etc. Written in C and assembler, taking advantage of many free replay routines.

Developed in 2005 and revitalized in 2013.

Download

mxPlay is a sourceforge.net project. You can download the latest release or use its web based svn browser.

Authors

mxPlay was programmed by Miro Kropacek aka MiKRO / Mystic Bytes in the winter of 2005. Skins and replay routines adaptation were done by -XI- / Satantronic. In 2013, I (Miro Kropacek) have revitalized the code, fixed various bugs and added XMP and ASAP replay routines.
